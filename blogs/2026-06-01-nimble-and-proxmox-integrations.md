While currently there is no official support Proxmox and support has been providing some generic Debian documentation upon request. A community member has been at work creating a integration plug-in to get the Nimble/Alletra arrays to talk with Proxmox. https://github.com/brngates98/pve-nimble-plugin In case anyone is looking to simplify the integration for test/lab environments.
